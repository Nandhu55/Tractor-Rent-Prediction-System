
# 🚜 Tractor Rent Prediction System

https://tractor-rent-prediction-system.streamlit.app/

A Machine Learning web application that predicts **tractor rental price per hour** based on:

- Tractor Type
- Working Hours
- Distance Covered

The application is built using **Python, Scikit-Learn, Pandas, and Streamlit**.

---

## 📌 Features

✔ Predict tractor rent instantly  
✔ Simple and interactive web interface  
✔ Machine Learning model using Linear Regression  
✔ Data visualization with charts  
✔ Easy deployment using Streamlit Cloud  

---

## 🧠 Machine Learning Workflow

1. Load dataset
2. Data understanding and cleaning
3. Feature selection
4. Train-test split
5. Train Linear Regression model
6. Model testing
7. Prediction
8. Build UI using Streamlit
9. Deploy on Streamlit Cloud

---


## 🛠 Technologies Used

- Python
- Pandas
- Scikit-Learn
